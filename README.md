
Histórias de um Banco


## 📒 Descrição
Este podcast consiste em contar histórias de pessoas que foram atendidas no banco, para resolver algum problema, mas aproveitam pra desabafar sobre sua vida.

## 🤖 Tecnologias Utilizadas
IA Generativa ChatGPT para roteirização e correção
Word para edição dos textos
Whatsapp para transformação do áudio em texto.
Elevenlabs para transformação de texto em áudio.

## 🧐 Processo de Criação
Utilização do chatGPT para aperfeiçoar os textos e melhor expressar as histórias a serem transmitidas,word para escrever os textos.
Elevenlabs para transformação do texto em áudio

## 🚀 Resultados
O podcast apresenta histórias que podem fazer rir, chorar e ter raiva.
